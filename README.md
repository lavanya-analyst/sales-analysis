This project analyzes sales data using Python.

##  Tools Used
- Python
- Pandas
- Matplotlib

##  Analysis Done
- Monthly Sales Analysis
- Category-wise Sales

##  Key Insights
- Month 8 has highest sales
- Month 6 has lowest sales
- Technology category performs best
- Sales vary across regions
